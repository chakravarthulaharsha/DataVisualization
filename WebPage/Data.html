Datasets Used :

    We have used the following datasets for our analysis:
  • Train.csv :
    Data type -> Table
    Attributes -> battery_power, blue, clock_speed, dual_sim, fc, four_g, int_memory, m_dep, mobile_wt, n_cores, pc,px_height, px_width, ram, sc_h, sc_w, talk_time, three_g, touch_screen, wifi, price_range
    No.of Records -> 2000
  • Mobilephone_prices (1).csv :
    Data type -> Table
    Attributes -> Phone_brand, Phone_price_kz, Phone_category, Operating_System, Phone_name, Phone_color, Phone_memory, Battery_life_in_mAh, Y_Shopping_sales, Y_Caala_sales
    No.of Records -> 85
  • smartphone_category.csv :
    Data type -> Table
    Attributes -> Product Name, Product URL, Brand, Sale Price, Mrp, Discount Percentage, Number Of Ratings, Number Of Reviews, Upc, Star Rating, Ram
    No.of Records -> 1513
  • Smartphone_Sales.csv :
    Data type -> Table
    Attributes -> Brand, 2021 Q1,2021 Q2, 2021 Q3, 2021 Q4, 2022 Q1,2022 Q2, 2022 Q3, 2022 Q4, 2023 Q1
    No.of Records -> 6
